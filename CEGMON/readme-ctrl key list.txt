CEGMON Control Key List
======================= CTRL Keys in standard mode (not EMACS option)

CEG = Standard CEGMON
SJG = RevD in place of Disk Bootstrap
XROM= XRevD at $8000

CTRL CEG SJG XROM   DESCRIPTION
---- --- --- ----   ------------------
A  - YES YES YES  - Edit Cursor LEFT
B  - 
C  - YES YES YES  - Break
D  - YES YES YES  - Edit Cursor RIGHT
E  - YES YES YES  - Toggle Edit Mode
F  - YES YES YES  - Edit Cursor DOWN
G  -
H  - 
I  - NO  NO  YES  - Increment Character Colour
J  - YES YES YES  - Cursor DOWN
K  - YES YES YES  - Cursor RIGHT
L  - YES YES YES  - Cursor HOME
M  - YES YES YES  - Return
N  - NO  YES YES  - Narrow Character Mode
O  -
P  - 
Q  -
R  - NO  YES YES  - RGB Mode
S  - YES YES YES  - Cursor UP
T  - 
U  - NO  YES YES  - Colour Fill
V  - NO  YES YES  - Video Mode (Composite B/W)
W  - NO  YES YES  - Wide Character Mode
X  - NO  NO  YES  - Toggle Screen Width 24/48 (32/64) column
Y  - NO  NO  YES  - Fill Screen
Z  - YES YES YES  - Clear Screen
